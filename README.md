### Configurações dos projetos

- Instalando nestjs de forma global: `$ npm install -g @nestjs/cli`
- Verificando ajuda do nest: `$ nest --help`
- Verificando versão do nest: `$ nest --version`
- Iniciando um novo projeto nest: `$ nest new nome_do_projeto`
- Entrando na pasta do projeto: `cd nome_do_projeto`
- Rodando o projeto nest: `npm run start`
